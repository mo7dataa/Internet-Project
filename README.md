# 🌐 Internet Usage Analysis Project

## 📌 Project Overview
This project focuses on analyzing global internet usage data using SQL and Excel.
The analysis explores internet penetration, broadband subscriptions, and mobile cellular subscriptions across different countries and regions over multiple years.

The goal of the project is to extract meaningful insights from the dataset and understand global internet usage trends.

---

## 📊 Project Objectives
- Analyze internet usage growth over time
- Compare internet usage between countries and regions
- Identify countries with the highest internet penetration
- Explore broadband and mobile subscription trends
- Create reusable SQL views for dashboard reporting

---
